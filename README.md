# ihcux-conectabus-cidade

ConectaBus - Protótipo de Baixa Fidelidade

Sobre o Projeto:
O ConectaBus é um aplicativo criado para unificar diferentes meios de transporte da cidade em um único lugar, facilitando a mobilidade urbana.

O usuário consegue:
Planejar rotas;
Visualizar lotação em tempo real;
Acompanhar o percurso no mapa;
Utilizar bilhete digital via QR Code;
Ver detalhes do veículo;
Receber alertas e notificações importantes.

Contexto de Uso:
O aplicativo foi pensado para pessoas em movimento, principalmente usuários com pressa, segurando mochila, sacolas ou utilizando o celular com apenas uma mão.

Exemplo:
Usuário correndo para não perder o ônibus ou metrô durante o horário de pico.

Decisões de UX
Algumas decisões tomadas no protótipo:
Tempo de chegada destacado para facilitar decisões rápidas;
Ícones grandes e fáceis de identificar;
Informações de lotação utilizando cores e linguagem simples;
Navegação intuitiva com poucos cliques;
Elementos acessíveis para uso com apenas um polegar.

Acessibilidade:
O protótipo busca ajudar usuários com mobilidade reduzida ou em situações de pressa através de:

Botões maiores;
Alto contraste nas informações importantes;
Identificação de acessibilidade nos veículos;
Fluxo simples e objetivo;
Redução de carga cognitiva.
